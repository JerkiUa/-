Каковы сроки проведения демонтажа?
+
Ответ на этот вопрос будет для каждого клиента разным.Так как у каждого разные потребности в демонтаже.

Например для снятия обоев может потребоваться один день, а для демонтажа бетонной стяжки(100м2) 2-3 дня

На полный демонтаж квартиры 70м2 уходит обычно неделя.
Как проходят демонтажные работы?
